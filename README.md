# Korobochka
Gaming console OS for esp32

Libraries need:
  1. Adafruit GFX
  2. Adafruit SSD1306
  3. Adafruit MLX90614
  4. AsyncElegantOTA
  5. Set of pre-installed esp32 libraries

Korobochka features 11 pre-installed apps:
  1. Space-Heroes
  2. Cat or Bread 2
  3. Tanchiki
  4. IR thermometer
  5. Math cheatsheet
  6. Piano (don't works currently)
  7. Portrait of Putin.png
  8. Polish cow.mp4
  9. Test app
  10. Beer's
  11. Settings:
      1. Piano settings
      2. Cat or Bread delay
      3. Medic thermometer
      4. ROM-tool
